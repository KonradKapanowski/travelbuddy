# TRAVEL BUDDY
Simple app to search attractions, hotels and food wherever you are.<br/>
You can also sort places by ratings.

## LINK:

https://gotravelbuddy.netlify.app/


## TECHNOLOGIES
React <br/>
Material UI

## API:
-GOOGLE MAPS API <br/>
